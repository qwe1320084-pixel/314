## Описание
Приложение на Python с графическим интерфейсом (tkinter), которое:
- Генерирует случайные цитаты из предустановленного или пользовательского списка.
- Сохраняет историю сгенерированных цитат в JSON.
- Позволяет фильтровать историю по автору и теме.
- Добавляет новые цитаты с проверкой на пустые строки.

## Установка и запуск
1. Убедитесь, что установлен Python 3.7+.
2. Скачайте файлы: `quote_generator.py`, `quotes.json` .
3. Запустите: python quote_generator.py
