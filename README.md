## Описание
Random Password Generator - это desktop-приложение с графическим интерфейсом для генерации безопасных случайных паролей. Программа позволяет настраивать длину пароля и типы используемых символов, сохраняет историю всех сгенерированных паролей и предоставляет возможность копировать пароли в буфер обмена.
